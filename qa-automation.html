Old Selenium way: page.find_element(By.CSS_SELECTOR, "#submit-btn") — breaks when CSS changes!

from playwright.sync_api import Page class LoginPage: def __init__(self, page: Page): self.page = page def login(self, user: str, pwd: str): self.page.goto("/login") self.page.get_by_label("Username").fill(user) self.page.get_by_label("Password").fill(pwd) self.page.get_by_role("button", name="Login").click()
🧠 Why Page Object Model?

When your login button CSS changes, you update ONE class. Without POM, you'd have to update 50 test files. POM = DRY tests! (Don't Repeat Yourself)

✅ CH 3 DONE
  • Page Object Model = one class per page, update once, fix everywhere
  • get_by_role / get_by_label = semantic selectors that survive UI changes
  • pytest fixtures auto-manage browser lifecycle — no manual cleanup!

from langchain_openai import ChatOpenAI from langchain_core.messages import SystemMessage, HumanMessage llm = ChatOpenAI(model="gpt-4o") def generate_test_from_story(story: str, html_snippet: str = "") -> str: msgs = [ SystemMessage("You are a senior QA engineer. Write pytest + Playwright tests."), SystemMessage("Use Page Object Model. Include assertions and edge cases."), HumanMessage(f"Story: {story}\nHTML: {html_snippet}\nWrite complete test file.") ] return llm.invoke(msgs).content story = "As a user, I want to add items to cart and see the total update." test_code = generate_test_from_story(story) print(test_code) # Writes complete pytest test!

When a selector breaks (button moved, ID changed, CSS class renamed), instead of failing, our agent takes a screenshot, sends it to GPT-4o Vision, asks "where is the Submit button?", and gets the new selector back automatically! 🪄

import base64 from langchain_openai import ChatOpenAI vision_llm = ChatOpenAI(model="gpt-4o") def heal_selector(page, broken_selector: str, element_purpose: str) -> str: # Take screenshot, ask AI to find the element screenshot = page.screenshot() img_b64 = base64.b64encode(screenshot).decode() response = vision_llm.invoke([ {"type":"image_url","image_url":{"url":f"data:image/png;base64,{img_b64}"}}, {"type":"text","text":f"Find '{element_purpose}'. Return only the new Playwright selector."} ]) return response.content.strip() # Returns: 'get_by_role("button", name="Submit Order")' — test continues!

  • heal_selector() returns a working Playwright selector automatically
